# AhaKey Community Configuration

This repository contains the organization-wide profile, community standards, and default contribution resources for AhaKey.

## What is this repository for?

This `.github` repository is used to manage shared community files for the AhaKey organization, including:

- organization profile README
- contribution guidelines
- support information
- security policy
- issue templates
- pull request templates

## About AhaKey

AhaKey is an open ecosystem built around:

- official hardware
- a shared BLE protocol
- open desktop software
- community-built clients, integrations, and workflows

The goal is to make AhaKey a flexible platform for AI coding workflows.

## Where to start

If you are visiting AhaKey for the first time:

- Read the organization profile
- Explore the `protocol` repository
- Try the `desktop` client
- Join Discussions
- Check contribution guidelines before opening issues or pull requests

## Notes

Files in this repository may be used as default community health files across repositories in the AhaKey organization.
