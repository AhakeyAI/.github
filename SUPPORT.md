# English

## Support

Thanks for using or building with AhaKey.

This document explains where to ask questions, report issues, and get help in the AhaKey ecosystem.

## Where should I go?

### For general questions
Use **Discussions** for:

- getting started questions
- workflow ideas
- setup help
- integration brainstorming
- community sharing

### For bugs and actionable problems
Use **Issues** for:

- reproducible bugs
- documentation mistakes
- platform-specific problems
- clear feature requests
- protocol-related issues

### For security issues
Please **do not** report security-sensitive issues publicly in Discussions or Issues.

Report them privately to:

**zhangxinyang@ahakey.cn**

## Before opening an issue

Please try to include:

- what you expected to happen
- what actually happened
- your platform or system version
- logs, screenshots, or reproduction steps when possible

This helps us and the community understand the problem faster.

## Community-maintained projects

Some tools and clients in the ecosystem may be community maintained.

If your issue is about a community project, please first check whether it should be reported to:

- the official AhaKey repository
- the community project repository itself

## Scope of official support

Official support is mainly focused on:

- official hardware
- official baseline software
- official documentation
- official protocol documentation

Community projects may be listed or showcased, but that does not automatically mean official support.

# 简体中文

## 支持与反馈

感谢你使用或参与 AhaKey 生态。

这份文档说明了在 AhaKey 社区里，遇到不同问题时应该去哪里提问、反馈或寻求帮助。

## 遇到不同情况，应该去哪里？

### 一般问题
以下内容建议发到 **Discussions**：

- 入门问题
- workflow 想法
- 使用与配置求助
- 集成思路讨论
- 社区分享

### bug 和可执行问题
以下内容建议发到 **Issues**：

- 可复现的 bug
- 文档错误
- 平台兼容问题
- 明确的功能请求
- 协议相关问题

### 安全问题
请**不要**在公开的 Discussions 或 Issues 中报告安全敏感问题。

请私下发送到：

**zhangxinyang@ahakey.cn**

## 提 issue 前建议附带这些信息

请尽量补充：

- 你原本期待发生什么
- 实际发生了什么
- 你的平台或系统版本
- 如果可以，附上日志、截图或复现步骤

这些信息会帮助我们和社区更快定位问题。

## 关于社区维护项目

生态中的部分工具或客户端可能由社区维护。

如果你的问题来自某个社区项目，请先判断应该提交到：

- 官方 AhaKey 仓库
- 对应的社区项目仓库

## 官方支持范围

官方支持主要聚焦在：

- 官方硬件
- 官方基础软件
- 官方文档
- 官方协议文档

社区项目可能会被收录或展示，但这并不自动代表官方支持。
